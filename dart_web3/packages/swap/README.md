# dart_web3_swap

DEX aggregation and swap execution for multiple protocols.

## Features

- **Protocol Support**: Uniswap v2/v3, Curve, Balancer integration.
- **Route Finding**: Find the most efficient swap path across DEXes.
- **Quote Extraction**: Get real-time pricing information.

## Installation

```yaml
dependencies:
  dart_web3_swap: ^0.1.0
```
