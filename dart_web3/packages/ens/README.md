# dart_web3_ens

Ethereum Name Service (ENS) resolution and reverse resolution.

## Features

- **Name Resolution**: Resolve `.eth` names to addresses.
- **Reverse Lookup**: Find names for a given Ethereum address.
- **Content Hash**: Retrieve IPFS/Swarm content hashes from ENS records.

## Installation

```yaml
dependencies:
  dart_web3_ens: ^0.1.0
```
