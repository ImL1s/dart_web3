# dart_web3_multicall

Efficient batch processing of multiple contract calls in a single transaction.

## Features

- **Batch Reading**: Aggregate multiple constant function calls.
- **Aggregation**: Supports Multicall2 and Multicall3 contracts.
- **Gas Optimized**: Reduce the number of RPC round-trips.

## Installation

```yaml
dependencies:
  dart_web3_multicall: ^0.1.0
```
