# dart_web3_debug

Debugging and tracing utilities for blockchain transactions.

## Features

- **Transaction Tracing**: Access Geth/Erigon style trace outputs.
- **State Overrides**: Simulate transactions with modified state.
- **Call Traces**: Hierarchical view of internal contract calls.

## Installation

```yaml
dependencies:
  dart_web3_debug: ^0.1.0
```
