# dart_web3_nft

NFT (Non-Fungible Token) services for ERC-721 and ERC-1155.

## Features

- **Metadata Parsing**: Fetch and parse NFT metadata (IPFS, HTTP).
- **Collection Discovery**: List NFTs owned by an address.
- **Interaction**: Easy minting, burning, and transfer tools.

## Installation

```yaml
dependencies:
  dart_web3_nft: ^0.1.0
```
