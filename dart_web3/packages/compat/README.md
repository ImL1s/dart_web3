# Web3 Universal Compat

A drop-in compatibility layer for `web3dart` users to easily migrate to the Web3 Universal SDK.

## Features

- Exact API match for `web3dart` core classes.
- Proxies calls to `web3_universal` implementation.

## Usage

```dart
import 'package:web3_universal_compat/web3_universal_compat.dart';
```
