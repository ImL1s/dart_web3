# dart_web3_reown

Integration with Reown (formerly WalletConnect v2) for mobile linking.

## Features

- **WC v2 Protocol**: Full support for WalletConnect v2 sessions.
- **UI Components**: (Optional) Ready-to-use Flutter connection dialogs.
- **Session Management**: Handle pairing, session authorization, and requests.

## Installation

```yaml
dependencies:
  dart_web3_reown: ^0.1.0
```
