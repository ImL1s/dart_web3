# dart_web3_bridge

Cross-chain bridging utilities for moving assets between different blockchains.

## Features

- **Protocol Support**: Integration with popular bridges (e.g., Hop, Stargate).
- **Transaction Tracking**: Monitor the status of cross-chain transfers.
- **Relayer Interaction**: Tools for interacting with bridge relayers.

## Installation

```yaml
dependencies:
  dart_web3_bridge: ^0.1.0
```
