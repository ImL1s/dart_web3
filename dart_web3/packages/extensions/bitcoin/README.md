# dart_web3_bitcoin

Bitcoin extension for the Dart Web3 SDK.

## Features

- **Address Generation**: SegWit (Bech32), Taproot, and Legacy support.
- **Transaction Creation**: UTXO management and transaction signing.
- **PSBT Support**: Partially Signed Bitcoin Transactions.

## Installation

```yaml
dependencies:
  dart_web3_bitcoin: ^0.1.0
```
