# dart_web3_solana

Solana blockchain extension for the Dart Web3 SDK.

## Features

- **Account Management**: Support for Solana keypairs and addresses.
- **RPC Client**: Interaction with Solana JSON-RPC API.
- **SPL Tokens**: Tools for managing Solana Program Library tokens.

## Installation

```yaml
dependencies:
  dart_web3_solana: ^0.1.0
```
