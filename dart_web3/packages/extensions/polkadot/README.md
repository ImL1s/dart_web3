# dart_web3_polkadot

Polkadot/Substrate extension for the Dart Web3 SDK.

## Features

- **Multi-Chain Support**: Interaction with Polkadot, Kusama, and parachains.
- **Metadata Parsing**: Support for Substrate runtime metadata.
- **Encoding**: SCALE codec implementation for Substrate types.

## Installation

```yaml
dependencies:
  dart_web3_polkadot: ^0.1.0
```
