# dart_web3_tron

Tron blockchain extension for the Dart Web3 SDK.

## Features

- **Protocol Support**: Support for TRC-10 and TRC-20 tokens.
- **Resource Management**: Manage Energy and Bandwidth on the Tron network.
- **Smart Contracts**: Interact with the Tron Virtual Machine (TVM).

## Installation

```yaml
dependencies:
  dart_web3_tron: ^0.1.0
```
