# dart_web3_ton

TON (The Open Network) blockchain extension for the Dart Web3 SDK.

## Features

- **Wallet Support**: v3, v4, and Highload wallet support.
- **BOC Handling**: Bag of Cells (BOC) serialization and deserialization.
- **Interaction**: Send messages and interact with TON smart contracts.

## Installation

```yaml
dependencies:
  dart_web3_ton: ^0.1.0
```
