# dart_web3_trezor

Trezor hardware wallet support for the Dart Web3 SDK.

## Features

- **Trezor Connect**: Support for communication with Trezor devices.
- **Secure Signing**: On-device signing for Ethereum and Bitcoin.
- **Passphrase Support**: Handle hidden wallets via Trezor's passphrase feature.

## Installation

```yaml
dependencies:
  dart_web3_trezor: ^0.1.0
```
