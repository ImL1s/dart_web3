# dart_web3_mpc

Multi-Party Computation (MPC) wallet support and threshold signatures.

## Features

- **TSS**: Threshold Signature Scheme implementation.
- **Key Sharding**: Split and manage private key shares.
- **Protocol Agnostic**: Flexible enough for various MPC network implementations.

## Installation

```yaml
dependencies:
  dart_web3_mpc: ^0.1.0
```
