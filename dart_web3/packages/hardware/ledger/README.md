# dart_web3_ledger

Ledger hardware wallet support for the Dart Web3 SDK.

## Features

- **Device Connection**: Connect via USB or Bluetooth (platform dependent).
- **Key Derivation**: Securely derive addresses from Ledger devices.
- **Transaction Signing**: Sign EIP-1559 and legacy transactions on-device.

## Installation

```yaml
dependencies:
  dart_web3_ledger: ^0.1.0
```
