# dart_web3_keystone

Keystone hardware wallet support via QR code communication.

## Features

- **QR Code Protocol**: Support for Keystone UR and BC-UR standards.
- **Air-Gapped Signing**: Sign transactions without physical connection.
- **PSBT Integration**: Compatible with Bitcoin and Ethereum signing flows.

## Installation

```yaml
dependencies:
  dart_web3_keystone: ^0.1.0
```
