# dart_web3_aa

ERC-4337 Account Abstraction support for Dart.

## Features

- **UserOperations**: Create and sign ERC-4337 User operations.
- **Bundler Client**: Communicate with ERC-4337 bundlers.
- **Paymaster Interaction**: Support for gas sponsorship.
- **Smart Accounts**: Base classes for implementing custom smart wallets.

## Installation

```yaml
dependencies:
  dart_web3_aa: ^0.1.0
```
